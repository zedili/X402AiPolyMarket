## 1. Cyber-Neural Interface (赛博神经接口)

### Design Movement
**Cyberpunk Futurism meets Data Visualization**: 结合赛博朋克的高科技感与现代数据可视化的清晰度，创造一个既神秘又实用的预测市场界面。

### Core Principles
1.  **Luminous Hierarchy (发光层级)**: 利用发光效果（Glow）的强弱来构建视觉层级，重要的预测数据和按钮应当"发光"，而背景则保持深邃。
2.  **Data Transparency (数据透明)**: 所有设计元素都应服务于数据的清晰展示，避免过度装饰干扰信息读取。
3.  **Fluid Intelligence (流动智能)**: 界面交互应当流畅如水，暗示背后AI算法的实时流动和计算。
4.  **Binary Contrast (二元对比)**: 强调YES/NO的二元对立，通过强烈的色彩对比（青色 vs 粉红/红色）来引导用户的决策。

### Color Philosophy
-   **Deep Void (深渊黑)**: `#0a0e27` 作为主背景，代表未知的未来和深邃的数据海洋。
-   **Neural Cyan (神经青)**: `#00d4ff` 代表理性的AI预测、科技和"YES"选项，传达冷静、准确的感觉。
-   **Pulse Pink (脉冲粉)**: `#ff0080` 代表人类的情感、不确定性或"NO"选项，与青色形成鲜明对比，制造视觉张力。
-   **Signal Green (信号绿)**: `#00ff88` 用于强调正向收益和确认操作。

### Layout Paradigm
**Modular Dashboard (模块化仪表盘)**: 摒弃传统的居中网页布局，采用类似交易终端的模块化布局。
-   **非对称平衡**: 侧重于信息流的展示，左侧可能作为导航或过滤器，右侧为主要内容区，或者采用全宽的卡片网格，模仿监控屏幕。
-   **HUD (Heads-Up Display) 元素**: 使用细线边框、角标装饰和半透明背景，营造HUD视觉体验。

### Signature Elements
1.  **Neon Borders (霓虹边框)**: 卡片和按钮使用带有微弱发光的边框，悬停时发光增强。
2.  **Glitch Text (故障文字)**: 在Logo或关键标题上使用轻微的故障艺术效果，强调"数字"属性。
3.  **Hexagonal Patterns (六边形纹理)**: 背景中融入低透明度的六边形网格，象征神经网络和区块链结构。

### Interaction Philosophy
**Tactile Feedback (触觉反馈)**: 所有的点击和悬停都应有即时的视觉反馈，如同按下物理开关或触摸全息屏幕。按钮点击时会有"压下"或"闪烁"的效果。

### Animation
-   **Pulse & Breathe**: 关键数据（如AI预测置信度）可以有轻微的呼吸效果。
-   **Scanlines**: 页面加载时模拟CRT屏幕的扫描线效果。
-   **Data Stream**: 数字变化时采用滚动数字动画，增加实时感。

### Typography System
-   **Display**: **Orbitron** 或 **Rajdhani** (Google Fonts)，用于标题和数字，具有科技感和方形结构。
-   **Body**: **Inter** 或 **Roboto Mono**，用于正文和数据标签，确保极高的可读性。

## 2. Ethereal Oracle (以太神谕)

### Design Movement
**Glassmorphism & Abstract 3D**: 结合毛玻璃效果和抽象3D元素，创造一个轻盈、梦幻且充满未来感的预测空间。

### Core Principles
1.  **Depth through Blur (模糊构建深度)**: 利用多层毛玻璃效果构建空间感，前景清晰，背景模糊。
2.  **Soft Gradients (柔和渐变)**: 使用大面积的柔和渐变色代替纯色，营造流动的氛围。
3.  **Minimalist Clarity (极简清晰)**: 减少线条和边框的使用，通过间距和色块区分内容。

### Color Philosophy
-   **Midnight Blue**: 深蓝紫色背景。
-   **Aurora Gradients**: 极光般的渐变色（紫、蓝、青混合）。
-   **Glass White**: 半透明白色用于卡片背景。

### Layout Paradigm
**Floating Cards (悬浮卡片)**: 内容以悬浮卡片的形式呈现，打破网格限制，布局更加自由流动。

### Signature Elements
1.  **Frosted Glass**: 广泛使用的毛玻璃背景。
2.  **Orbital Shapes**: 背景中漂浮的圆形或球体元素。

### Interaction Philosophy
**Smooth Flow**: 交互如丝般顺滑，强调过渡的自然性。

### Animation
-   **Float**: 元素轻微浮动。
-   **Fade & Slide**: 内容切换时的淡入淡出和滑动。

### Typography System
-   **Display**: **Space Grotesk**。
-   **Body**: **DM Sans**。

## 3. Brutalist Data (粗野数据)

### Design Movement
**Neo-Brutalism**: 强调原始、粗犷和高对比度的设计风格，回归互联网早期的纯粹功能主义。

### Core Principles
1.  **Raw Structure (原始结构)**: 暴露网格和布局结构，使用粗黑边框。
2.  **High Contrast (高对比)**: 黑白为主，辅以高饱和度的原色。
3.  **System Fonts (系统字体)**: 使用默认或类似终端的字体。

### Color Philosophy
-   **Stark Black & White**: 极致的黑白对比。
-   **Neon Yellow/Green**: 高亮色用于强调。

### Layout Paradigm
**Strict Grid (严格网格)**: 明显的网格线分割各个区域。

### Signature Elements
1.  **Thick Borders**: 粗边框。
2.  **Hard Shadows**: 硬阴影，无模糊。

### Interaction Philosophy
**Direct & Snappy**: 交互直接，无过渡动画，强调响应速度。

### Animation
-   **None or Minimal**: 极少使用动画，或仅使用简单的位移。

### Typography System
-   **Display & Body**: **Courier Prime** 或 **JetBrains Mono**。
